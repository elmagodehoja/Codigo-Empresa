# Codigo-Empresa
Instrucciones para el colaborador
Para completar este ejercicio con éxito, sigue estos pasos:


Clona este repositorio en tu máquina local utilizando la terminal.

Modifica el archivo colaboradores.txt: Añade tu nombre y apellidos al final de la lista.

Actualiza el archivo script.py: Añade una línea de código simple (por ejemplo, un print con un mensaje de saludo).

Guarda tus cambios: Realiza un commit descriptivo y haz push de tus cambios al repositorio remoto.

Finaliza: Verifica que tus cambios aparecen correctamente en la versión web de GitHub.
